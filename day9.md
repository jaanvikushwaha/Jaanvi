# Day 8 – NutriScope Nutrition Dashboard

## 📌 Project Overview
NutriScope is a single-file HTML nutrition dashboard that helps users track daily nutrition, monitor macro and micronutrient intake, visualize health data, and receive personalized nutrition recommendations.

---

## 📂 Files Included

- `NutriScope_MVP.html`
- `NutriScope_Enhanced.html`

---

## 🔄 Comparison Notes

| MVP | Enhanced Version |
|------|------------------|
| Basic nutrition tracking | CSV Upload Support |

| Basic dashboard | Interactive charts |

## 📚 Key Learnings

- Built a complete nutrition dashboard using HTML, CSS, and JavaScript.
- Learned to organize food nutrition data efficiently.
- Implemented macro and micronutrient calculations.
- Improved data visualization with Chart.js.
- Designed a responsive and modern user interface.
- Enhanced user experience through personalized recommendations and meal planning.
